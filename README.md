## Анализ работы крупного спортивного магазина.

### Задачи анализа :

- проверка эффективности проведенной маркетинговой кампании с использованием А/В-теста;

- выявление кластеров клиентов для предложения более таргетированной рекламы;

- прогноз успешности планируемых мероприятий.

### Выводы:

- увеличение средней стоимости потребительской корзины и количества товаров позволяют считать проведенную кампанию вполне успешной;

- кластеризация клиентов позволила выделить группы для персональных рекомендаций, а не общую скидку;

- согласно прогнозу 76% клиентов будут участвовать в будущей маркетинговой кампании.

Общая рекомендация - ограничить количество цветов в описании продукта и повысить качество заполняемых данных
